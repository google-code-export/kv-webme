{{PRODUCTS_BUTTON_ADD_TO_CART}} Button for adding the product to a cart. Optional parameters:
  • text: what text should be shown in the button. Default is "Add to Cart".
  • redirect: whether to load the same page again ("same"), or redirect to basket ("checkout").
{{PRODUCTS_BUTTON_ADD_MANY_TO_CART}} Button and input box for adding many items to the cart. Optional parameters:
  • text: what text should be shown in the button. Default is "Add to Cart".
  • redirect: whether to load the same page again ("same"), or redirect to basket ("checkout").
  • type: what type of interface to use. default is "input". options are: "input", "select".
  • min: minimum to allow to add from this button.
  • max: maximum to allow from this button. If the "type" is "select", then this will default to 50.
